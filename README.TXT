HospitalUEx es un proyecto de C++ que simula la gestión de un hospital.  
A partir de un archivo .csv se carga una serie de pacientes, detecta errores y realiza una serie de busquedas y estadísticas sobre las estructuras de datos usadas para contener los datos.
El objetivo del proyecto es profundizar en las diferentes estructuras de datos que se utilizan en lenguajes de programación, como crearlas, usarlas y mantenerlas.
Esta es la tercera entrega del proyecto.
Este proyecto ha sido creado para la asignatura de Estructuras de Datos y de la Información del Grado de Ingeniería en Ingeniería del Software de la UEx año 2022/2023.  

Creado por José Manuel de Torres Domínguez.  

En este repositorio se encuentra un proyecto de Eclipse for C++.  
Es necesario tener las herramientas necesarias instaladas, como por ejemplo el compilador de C.
